Detailed Description

Auto-publishes field telemetry for balls and robot poses.

Publishes structured pose arrays to AdvantageScope-friendly topics:

  • /jsim/world/ballPoses
  • /jsim/world/robotPoses

The publisher registers itself with the world step loop, so calling PhysicsWorld#step() refreshes the topics automatically.
